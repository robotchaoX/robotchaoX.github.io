vim 编辑器

vim 是从 vi 发展出来的一个文本编辑器，支持 代码补全、编译 及 错误跳转 等方便编程的功能特别丰富，在程序员中被广泛使用，被称为 编辑器之神，和Emacs并列成为类Unix系统用户最喜欢的编辑器。

vim 的特点：

  • 没有图形界面 的编辑器
  • 只能是编辑 文本内容，不能对字体、段落进行排版
  • 不支持鼠标操作
  • 没有菜单
  • 只有命令
  • 在使用 vi 命令时，注意 关闭中文输入法
